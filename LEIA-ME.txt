=====================================================
      SISTEMA DE SUPORTE PROFISSIONAL - Gxzinkk_
=====================================================

Obrigado por adquirir nosso sistema! Siga os passos 
abaixo para configurar e ligar o seu bot de forma 
manual e segura.

1. REQUISITOS:
   - Ter o Node.js instalado (versão 16.x ou superior).

2. INSTALAÇÃO DAS DEPENDÊNCIAS:
   - Abra o terminal (Prompt de Comando ou PowerShell) na pasta do bot.
   - Digite o comando: npm install
   - Aguarde a conclusão da instalação da biblioteca discord.js.

3. CONFIGURAÇÃO (config.json):
   - Abra o arquivo 'config.json' com um editor de texto.
   - 'token': Insira o Token do seu bot (Discord Developer Portal).
   - 'owners': Insira o seu ID entre aspas. Ex: ["123456789"].
   - 'cargoStaff': ID do cargo que gerencia os tickets.
   - 'categoriaTickets': ID da categoria onde os canais serão criados.
   - 'canalLogs': ID do canal onde as ações serão registradas.

4. LIGANDO O BOT:
   - No terminal aberto na pasta do bot, digite: node index.js
   - Se aparecer "Sistema Gxzinkk_ Online", o bot já está operando.

5. COMANDOS PRINCIPAIS (Slash Commands /):
   - /setup: Configura o painel de tickets (Título, Descrição, Banner).
   - /config-bot: Altera Nome e Avatar do bot em tempo real.
   - /add e /remove: Adiciona ou remove membros de um ticket aberto.

⚠️ IMPORTANTE: No Developer Portal (aba Bot), ative as 3 chaves de 
'Privileged Gateway Intents' para que os comandos funcionem corretamente!
