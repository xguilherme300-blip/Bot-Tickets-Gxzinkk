=====================================================
      SISTEMA DE SUPORTE PROFISSIONAL - Gxzinkk_
=====================================================

Obrigado por adquirir nosso sistema! Siga os passos
abaixo para ligar o seu bot em menos de 5 minutos.

1. CONFIGURAÇÃO (config.json):
   - Abra o arquivo 'config.json' com o Bloco de Notas ou VS Code.
   - 'token': Coloque o token do seu bot (obtido no Discord Developer Portal).
   - 'owners': Coloque o SEU ID entre aspas. Ex: ["123456789"].
   - 'cargoStaff': ID do cargo que poderá ver os tickets e usar comandos de Staff.
   - 'categoriaTickets': ID da categoria onde os canais de ticket serão criados.

2. LIGANDO O BOT:
   - Clique duas vezes no arquivo 'INSTALAR.bat' para baixar as bibliotecas.
   - Clique duas vezes no arquivo 'LIGAR.bat' para iniciar o bot.

3. COMANDOS PRINCIPAIS:
   - /setup: Use para criar o seu painel personalizado (Título, Banner, Cor).
   - /config-bot: Use para mudar o Nome e a Foto do bot direto pelo Discord.
   - /ping: Verifica se a conexão do bot está rápida.

⚠️ IMPORTANTE: No Developer Portal do Discord, ative as 3 CHAVES 
de 'Privileged Gateway Intents' na aba "Bot" para os comandos funcionarem!
